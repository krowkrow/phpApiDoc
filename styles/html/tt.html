POST - 新增权限组 - /permgroup/newrec

/permgroup/newrec

注销登录

输入参数

参数 类型 必须 默认值 描述
group_name string 是 权限组名称，限长255字符
perm string 是 权限ID，多个英文逗号分隔
notes string 是 权限组备注，限长255字符

返回

{
    "code":"0",
    "msg":"成功",
    "data": "登出成功"
}

纯净json

{
    "code":"0",
    "msg":"成功",
    "data": "登出成功"
}

POST - 修改权限组 - /permgroup/modifyrec

/permgroup/modifyrec

修改一条权限组记录

输入参数

参数 类型 必须 默认值 描述
perm_group_id int 是 权限组id
group_name string 是 权限组名称，限长255字符
perm string 是 权限ID，多个英文逗号分隔
notes string 是 权限组备注，限长255字符

返回

{
    "code":"0",
    "msg":"成功",
    "data": "登出成功"
